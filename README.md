# README — Forge \& Fortune

# Concept

# 

# Forge \& Fortune est un jeu de plateau solo au tour par tour inspiré de Loop Hero. Le joueur incarne un forgeron un peu trop sûr de lui, perdu dans un monde médiéval sombre où tout semble aller de travers… mais rarement de manière sérieuse. Entre créatures absurdes, PNJ douteux et situations improbables, l’aventure oscille constamment entre danger réel et humour grinçant à la Donjons de Naheulbeuk.

# 

# Le but est simple : survivre et compléter trois tours complets du circuit. En pratique, chaque case réserve une surprise, souvent mauvaise, parfois lucrative, toujours imprévisible.

# 

# Boucle de jeu

# 

# À chaque tour :

# 

# Le joueur lance un dé (1 à 6)

# Le personnage avance automatiquement

# L’événement de la case se déclenche

# 

# Types de cases

# 

# Dialogue

# Rencontre avec un PNJ : marchand louche, noble ruiné, gobelin philosophe…

# Chaque dialogue propose des choix avec conséquences :

# 

# Gain ou perte de Gold

# Acquisition ou utilisation de clés

# Déblocage de flags narratifs persistants

# 

# Récompense

# Un moment de répit (relatif) : découverte de ressources ou d’objets, principalement du Gold.

# 

# Piège

# Mauvaises décisions passées, routes mal fréquentées ou simple malchance. Le joueur perd du Gold de façon aléatoire.

# 

# Mini-jeu — Cache-Cache

# Transition vers un niveau dédié :

# 

# Mode 0 : le joueur se cache, une IA cherche

# Mode 1 : l’IA se cache, le joueur cherche

# Durée : 20 secondes

# Résultats :

# Réussite : +25 Gold

# Échec : -15 Gold

# 

# Le résultat est stocké et appliqué au retour sur le plateau.

# 

# Progression \& Conditions

# Victoire : compléter 3 tours complets du circuit

# Défaite : tomber à 0 Gold

# 

# (Chaque tour devient progressivement plus risqué à mesure que les ressources fluctuent et que les choix passés influencent les événements.)

# 

# Direction artistique \& ton

# 

# L’univers mélange :

# 

# Ambiance médiévale sombre et légèrement crasseuse

# Personnages absurdes, souvent incompétents ou cyniques

# Situations dramatiques détournées par l’humour
